# HardClipOptions

Hard clipping configuration.

```ts
interface HardClipOptions {
    drive: number;
    gain: number;
    strictMode: StrictMode;
}
```

## About
Defines parameters for the hard clipping effect.

## Properties
- `drive`: `number` - Drive amount.
- `gain`: `number` - Output gain.
- `strictMode`: `StrictMode` - Validation behavior.
