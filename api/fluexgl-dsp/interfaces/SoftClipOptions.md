# SoftClipOptions

Soft clipping configuration.

```ts
interface SoftClipOptions {
    drive: number;
    gain: number;
    strictMode: StrictMode;
}
```

## About
Defines parameters for the soft clipping effect.

## Properties
- `drive`: `number` - Drive amount.
- `gain`: `number` - Output gain.
- `strictMode`: `StrictMode` - Validation behavior.
