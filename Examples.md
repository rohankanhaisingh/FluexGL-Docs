### About 
DSP Mixer is an open-source, browser-based digital mixer application designed to demonstrate advanced real-time audio routing and digital signal processing, using FluexGL DSP. The application allows you to route multiple audio clips into individual mixer channels, where each channel can be independently controlled and processed, similar to a professional digital mixing console.

Each channel supports common mixer functionality such as volume control, muting, and effect chains, enabling precise control over how audio is shaped and combined. Effects are applied in real time using a DSP-driven audio pipeline, making it possible to experiment with audio processing concepts directly in the browser. The overall goal of DSP Mixer is to provide a realistic and interactive mixing experience while remaining lightweight, transparent, and easy to extend.

DSP Mixer also serves as a technical showcase for the underlying DSP architecture, illustrating how audio clips, channels, and effects can be composed into a flexible routing graph. Because the project is fully open-source, it can be used as a learning resource, a testing environment for new DSP features, or a foundation for more advanced audio applications such as sequencers, DAWs, or game-audio tools.
